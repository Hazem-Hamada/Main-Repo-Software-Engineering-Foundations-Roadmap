# 📘 Algorithms and Problem Solving – Level 1
> **"Learn How to Think Before You Learn How to Code"**

This repository marks the completion of the second foundational step in my software engineering roadmap. This course focused entirely on the core of programming: **Problem Solving**.

---

## 🗝️ Course Overview
The philosophy of this course is that "Programming is not a language; a language is only a tool." I spent this stage learning how to analyze complex problems, break them down into manageable steps, and design logical solutions without writing a single line of code.

## 📖 Key Topics Covered
* **Algorithmic Thinking:** Understanding how to approach any problem logically and systematically.
* **Problem Analysis:** Identifying Inputs, Processes, and Outputs (IPO).
* **Visual Logic (Flowcharts):** Designing solutions using flowcharts to visualize decision-making and loops.
* **Mental Modeling:** Solving 50 diverse logical problems using only "paper and pen" to strengthen the mental execution of logic.

## 🎯 Learning Outcomes
* Mastered the ability to think like a problem solver.
* Built a solid foundation in structural logic, independent of any programming syntax.
* Prepared for the transition into coding (C++) with a clear, algorithmic mindset.
